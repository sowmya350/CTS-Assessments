# Exercise 3: Sorting Customer Orders

## Understand Sorting Algorithms
Sorting helps arrange data in a meaningful order. For customer orders, sorting by total price allows high-value orders to be processed first.

## Setup
No external libraries are required. A standard Python environment is enough.

## Implementation
The solution uses Python's built-in sorting, which applies an efficient comparison-based algorithm.

## Analysis
- Time complexity: O(n log n)
- Space complexity: O(n) in the worst case

The implementation is available in [sorting_customer_orders.py](sorting_customer_orders.py).
